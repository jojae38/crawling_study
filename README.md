CRAWLING STUDY

## 소개
통합적 GUI 틀을 만들기 위해 생성됨

## 주요 기능
- 라이브러리 자동 설치
- 실행 파일 자동 생성
- 깃허브 기반 업데이트 기능
- 로그인 기능
- 인증 코드 발급 기능
- 광고 기능

## 설치 및 실행 방법

### 설치
```bash
git clone https://github.com/jojae38/crawling_study.git
cd crawling_study
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 실행
```bash
cd crawling_study
python main.py
```
